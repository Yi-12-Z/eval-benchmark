# LLM 代码任务表现对比基准测试

## 项目简介

本项目旨在建立一个客观、量化的评估体系，用来系统性地对比不同大语言模型在代码生成、错误调试、代码解释等方面的表现。目标是找到最适合特定开发流程的模型，为技术选型提供数据支持。

## 核心方法与工具

整个评估流程基于 [ChainForge](https://chainforge.ai/)构建，这是一个强大的AI模型评估框架。我通过它设计了一套标准化的Prompt（提示词）和评估任务集。
